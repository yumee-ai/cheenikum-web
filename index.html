How it works?

Feature Description
Extension Activation As soon as the user visits a "XYZ" website, the extension is activated.
Data Protection Data in your clipboard is analyzed for sensitive information.
Data Analysis Analysis is done locally, using the gemini Nano model hence no data is sent to cloud!
Data Masking The sensitive data in your clipboard is masked as soon as the current tab is changed.
Data Protection The data send to the "XYZ" app is sanitized, so that it does not contain any sensitive information.
Data Un-Masking Once you copy back the data, it is unmasked in your clipboard directly, for your convenience.

Questions & answers

What does this extension do? +
This extension removes sensitive information from your clipboard, so that it is not accidently exposed to AI tools.
How do I install and use it? +
Install it from the Chrome Web Store . Once added, click on the extension icon in your browser’s toolbar to customize the extension.
Is the extension free? +
Yes, the extension is free to use. However, you can upgrade for unlimited usage. Night's Watch Plus version.
How can I add more websites to be protected? +
You can add more websites to be protected by clicking on the extension icon in your browser’s toolbar and then clicking on the "Add Website" option.
How can I add more tags? email, credit card, etc. +
You can add more tags by clicking on the extension icon in your browser’s toolbar and then clicking on the "Add Tag" option.
How can I get help or suggest features? +
Contact us through the help center . We’re happy to help and always open to suggestions!
